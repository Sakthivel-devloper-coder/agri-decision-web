# AgriDecision Web (scaffold)

Minimal Angular 17 standalone scaffold for the AgriDecision dashboard.

Quickstart

1. Install Node.js (16+ recommended) and Git.
2. Install Angular CLI globally if you want: `npm i -g @angular/cli` (optional)
3. From the project root:

```powershell
npm install
npm run start
```

Notes
- This scaffold contains a minimal dashboard feature with mocked data and core/shared examples.
- You will still need to run `npm install` locally — the environment here cannot run npm.
